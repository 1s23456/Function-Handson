  Ans :-            function add(){

                 console.log(c)
               }
              add();
Explanation -

This code defines a function named add. Within this function, two variables are declared, a and b, and assigned values of 2 and 3 respectively. Q4: Print output:
                  var x = 21;
                  var girl = function(){
                            console.log(x);
                            var x =20;
                  };
                  girl()
               
  Ans:   OUTPUT is:- "undefined"
Explanation :

The code defines a global variable x with the value 21. Then, it defines a function girl that logs a variable x to the console. Inside the function, a new variable x is declared with the value 20, but it only has function scope and is not the same as the global variable x. When the function is called and logs the variable x, the local variable x has not yet been assigned a value, so it logs undefined to the console.

Q7 : Write the function that accepts parameter n and returns factorial of n.

      ANS:   function factorial(n){
                          let result = 1 ;
                          for(let i = 1;i<=n ; i++){
                              result = result * i;
                        }
                          return result ;
                    }
                    console.log(factorial(5));
             
              OUTPUT IS : 120
Explanation:

This function uses a for loop to calculate the factorial of n. It starts with a result of 1 and then multiplies it by every number from 1 to n.

                  function FindSum(a,b){
                    return a + b;
                  }
                  function DisplayData(data,batch){
                    console.log(`i am ${data} and My batch is EA${batch}`)
                  }
                  DisplayData("Prepbytes",FindSum(10,9));
             
OUTPUT IS : i am Prepbytes and My batch is EA19
Explanation:

The code has two functions: "FindSum" and "DisplayData". The "FindSum" function takes two parameters "a" and "b" and returns the sum of their values using the addition operator. So if I call the function with the arguments "10" and "9", it will return the value as "19". The code calls the "DisplayData" function with the arguments "PrepBytes" and the result of calling "FindSum" with the arguments "10" and "9". This means that the "FindSum" function will return the value "19", and that value will be passed as the second argument to "DisplayData". As a result, the final output displayed in the console will be "i am from PrepBytes and My batch is EA19". 

Q2: Guess the Output :
               Abc();
               const Abc = function(){
                    let value = 20;
                    console.log(value);
               }
             
 OUTPUT IS : Error
Explanation:

This code will result in an error because the function "Abc" is declared using a function expression which is assigned to a variable using the "const" keyword. When the program starts running, the "Abc()" function is called before it is declared. However, function expressions are not hoisted like function declarations, meaning that they cannot be called before they are defined. So it will through an error. 

Q3 : Guess the Output:
                var a = 10;
                (function (){
                    console.log(a);
                    var a =20;
                })();
             
 OUTPUT IS : Undefined
Explanation:

The output of this code is "undefined" because of variable hoisting, which causes the declaration of "a" to be moved to the top of the function scope, but its value is not assigned until later in the function.
